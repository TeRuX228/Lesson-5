GIT flow

  • GIT pull-для того, чтобы получить изменение с репозитория
  • GIT add-добавить изменения, которые сделали локально
  • GIT commit-m "commit text"-подписать их
  • GIT push origin branchame-запушить на git сервер
  • GIT clone-для того чтобы "клонировать" репозиторий
  • GIT checkout-b branchname
